# Literature Review Protocol

Цел: систематичен обзор ~200 peer-reviewed статии, минимум 40 страници.

## Research Questions
- RQ1: e-nose хардуер/протоколи и основни проблеми (drift, T/RH, калибрация)
- RQ2: Feature extraction и ML pipelines за machine olfaction
- RQ3: Neuromorphic olfaction (spike encoding, SNN)
- RQ4: FPGA реализации за SNN/NN (архитектури, fixed-point, памет, throughput, power)

## Sources
IEEE Xplore, ACM DL, Elsevier/ScienceDirect, Springer, MDPI Sensors, Nature, arXiv (само като допълнение).

## Inclusion/Exclusion
- 2010–2026 (по-стари само “класики”)
- peer-reviewed (основно)
- съдържа експеримент/архитектура/протокол, не само общи мнения

## Data extraction table (минимум колони)
- Domain: e-nose / neuromorphic / FPGA
- Sensors (type/count), sampling
- Dataset size/classes
- Methods (features, model)
- Metrics (acc/F1, drift handling)
- Hardware metrics (LUT/BRAM/DSP, latency, power)
